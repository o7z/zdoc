# 搜索

## 触发

- macOS：`⌘ + K`
- 其他系统：`Ctrl + K`

## 它在搜什么

只搜**侧边栏条目本身**：title 与路径。所以一定能找到你建过 `_meta.yaml.pages` 条目的页面，但**搜不到正文**。

## 为什么不索引正文

- zdoc 的"零配置"承诺意味着不引入额外的索引/检索基础设施
- 大部分文档库 < 200 页，标题搜索就够用
- 真要全文搜索时，扔 ripgrep 进文档目录就行

## 键盘交互

| 键           | 行为                |
|--------------|---------------------|
| 输入字符     | 实时模糊匹配        |
| `Enter`      | 跳到第一个结果      |
| `Esc`        | 关闭弹窗            |
| 鼠标点击     | 跳到对应结果        |

## 已知限制

- 不支持中文分词（用 `includes` 做子串匹配，所以"会话"能匹配到"会话与持久化"，但"持久化会话"匹配不到反向词序）
- 不显示来源目录，列表里只有标题与 URL
