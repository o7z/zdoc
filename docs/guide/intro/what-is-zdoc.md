# 什么是 zdoc

zdoc 是一个**零配置**的 Markdown 文档站。把一个目录交给它，它就给你一套带：

- 自动侧边栏
- 右侧目录（h1–h3）
- 可选的密码保护与会话持久化
- 暗色模式 + Ctrl/Cmd-K 搜索

的现代化文档站点 —— 不需要再选静态站点生成器、不需要写配置。

## 适用场景

- 给团队内部分享一份会持续更新的设计/接口文档
- 写一些只想**自己看**的笔记，但又不想用 Notion / Obsidian 的同步
- 给客户私享一份带访问密码的资料库

## 不适用场景

- 公开博客（zdoc 不生成 RSS/sitemap，也不搞 SEO）
- 需要复杂主题或 React 组件的站点 —— 那是 VitePress / Docusaurus 的领域
- 想要 i18n 路由 —— 目前不支持

## 它的边界

zdoc 只做"把 Markdown 渲染成网页"这件事，外加一点点鉴权和元数据。设计目标是：

> 五分钟内从零跑起来，半年后还能让人看懂代码。
